## 在本地端運行

若想在本地端執行本系統，請依照以下步驟：

```bash
conda create -n fin-ins python=3.11 -y
conda activate fin-in
conda install -c conda-forge jupyter -y

#啟動jupyter
conda activate fin-in && jupyter notebook
```
